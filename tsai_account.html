How to Create and Manage Your TSAI Account

1. Create TSAI Account Using Signup Code

  • Navigate to https://www.techstarvoip.com/#/login and hit Register in the top right.
  • Enter your phone number and hit Send to get a verification code. Enter the code in the box.
  • Choose a password.
  • Verify your password.
  • Paste the invitation code into the box.
  • Click Register.

2. Authenticate Yourself

  • Navigate to TSAI Authentication.
  • Enter your details and upload a picture of your photo ID (front and back).
  • Click Submit.

3. Set a Transaction Password

  • Navigate to TSAI Security Center.
  • Click the menu button in the top left once more.
  • Select Security Center.
  • Click Transaction Password.
  • Enter your new password twice.
  • Press Send to receive a verification code on your phone number. Enter the code in the box.
  • Press Submit changes.

4. Link Your Coinbase USDT Address

  • Navigate back to TSAI Address Book.
  • Hit Add in the top left corner to add an address.
